C语言

PE格式

  • Win：.exe/ .dll/ .sys
    Linux：.elf

  • 构成

    • DOS部首
      e_magic(MZ): “0x5a4d”
      DOS Stub:”this program Cannot be run in DOS mode”(起兼容作用)
      e_lfanew:指向真正PE文件头“0x3c”

    • PE文件头

      ​ 由”PE\0\0”开头”0x50 0x45 00 00”
      ​ MAGE_OPTIONAL_HEADER32和IMAGE_FILE_HEADER定义PE的数据
      ​ 数据目录表中有PE使用的导入表（引用的函数）与导出表（程序本身定义的函数）

    • 块表

      ​ 指向块

    • 块

      ​ 用来存放信息与代码等

    • 调试信息

      ​ 用于调试

  • **Imagebase:**基地址,磁盘中的文件加载到内存当中的时候可以加载到任意位置，而这个位置就是程序的基址
    VA: 虚拟内存地址,PE 文件被操作系统加载进内存后的地址。

    RVA: PE文件的相对虚拟地址,PE文件中的数据、模块等运行在内存中的实际地址相对PE文件装载到内存的基址之间的距离
    VA=ImageBase+RVA

进制

C：
十六进制：0x开头
二进制：无单独二进制表示方式

汇编语言：
十六进制：H结尾(a=10,……f=15)
八进制 ：O结尾
十进制 ：D结尾
二进制 ：B结尾

  • 进制转换：

​

十六进制：0x11a=1*16^2+1 16^1+a16^0=282

八进制：0127=1*8^2+2 *8^1+7=87

二进制：00101101=1*2^5+1 *2^3+1 *2^2+1=45

编码

  • 原码：数值前增加一位符号位(0为正，1为负),符号位无法参与运算
    反码：正整数为其自身，负整数通过对其绝对值逐位求反来求，但会出现（+0），（-0）
    补码：正数为本身，负数为绝对值求反加1，0补码为0,用符号位扩充

存储方式

  • 低位优先：x86

浮点数

单精度浮点数：float，占4个字节 ，数字后加“f”
阶码8位，尾数23位

双精度浮点数：double，占8个字节
阶码11位，尾数52位

数据类型

字符：char(ASCII字符) 1个字节/wchar_t（UNICODE字符，字符前加L)2或4个字节

整型：short2个字节/int4个字节/long4或8个字节/long long/_int64

实数单精度：float4个字节

实数双精度：double8个字节

有符号和无符号：signed/unsigned,(signed)int/unsigned int

布尔类型：bool，0/1（true/false）1个字节（C99之后，或C++）
BOOL4个字节（windows）

Ascall编码

字符转整数：c-‘0’//‘5’–’0’=5
整数转字符：i+’0’//‘5’+’0’=’5’

char c=’M’;
c+’a’-‘A’;// 大写转小写
char ch=‘y’；
ch+’A’-‘a’;//小写转大写

表示的数的范围

char:[-128，+127] / [0，255]
short:[-32768,32767] / [0,65535]
int/long:[–2147483648,2147483647] / [0,4294967295]
float:[-3410^38,3.410^38]
double:[-1.710^-308,1.710^308]
有符号范围 [ -2 ^(n-1),2^(n-1)-1 ]
最大向上溢出1变最小，最小向下溢出1变最大

输入输出

使用scanf函数接收输入（由于安全问题可使用scanf_s代替）

在连续两个接收数据时,回车也会当做一个字符，导致后面的接收出错，所以应该在每次scanf后输入：

scanf输入时使用“空格”，空格后不显示输出

可以使用其他函数

也可以用_getch()代替（可以输出空格）

  • %
    • %.[数字]可以控制浮点数后面有几位小数
    • %d/i int 接受整数值表示为有符号十进制
    • %o unsigned int 无符号8进制（不输出前缀0）
    • %u unsigned int 无符号10进制整数
    • %x/X unsigned int 无符号16进制整数，x对应abcdef，X对应ABCDEF（不输出前缀0x）
    • %f(lf) float(double) 单精度浮点数用f，双精度用lf（尤其scann不能混用）
    • %e/E double 科学计数法表示，e大小写表示输出时大小写
    • %g/G double 使用以上两种中最短的形式，大小写同上
    • %c char 字符型，输入数字按照ASSCII码转换字符
    • %s/S char*/wchar_t* 字符串（输出至空字符）
    • %p void* 16进制输出指针
    • %n int* 到此字符前输出字符个数（不输出文本）
    • %% 无输入 不转换，输出%
    • %m 无 打印出错内容

优先级从上到下

“( )”改变优先级和结合律，不确定优先级时加括号

“!”逻辑非：!0→1 ![非零]→0 !false(0)→true(非零)

i++/++i：把i加一
i–/–i：把i减一

  •         在c++里，i是一个对象，++i/--i比i++/i--效率高

  •         在复合表达式或赋值语句里

  • 函数中
    int i=0;

    printf(“i:%d\n”,i++); →0（先输出i）

    i=0;
    printf(“i:%d\n”,++i); →1（先+）

    i=0
    printf(“%d %d\n”,i++,++i); 不确定，看编译器先用哪个

浮点数和0比较 (不能直接和0比较，看绝对值是否小于一个特别小的小数)

&&：且

+= -= *= /= %=

逗号运算符优先级最低

贪心算法：编译器将程序分解成符号时，从左到右一个字符一个字符读入，如果可能组成一个符号，那么再读入下一个字符，判断已经读入的两个字符组成的字符串是否可能是一个符号的组成部分；如果可能，继续读入下一个字符，重复该操作直到读入字符组成字符串不可能成为一个有意义的符号
